## 丨安装提醒

> **注意：该插件为 [早柚核心(gsuid_core)](https://github.com/Genshin-bots/gsuid_core) 的扩展，具体安装方式可参考上方安装文档**
>
> **运行环境要求 Python `3.11+`**
>
> **如果已经是最新版本的 `gsuid_core`，可以直接对 bot 发送 `core安装插件NTEUID`，然后重启 Core 以应用安装**
>
> 首次使用登录功能前，请确认 Core 的 Web 服务可以被浏览器访问
>
> 如需手动指定登录地址，可在配置中填写 `NTELoginUrl`
>
